Nav and Tabs

El componente base .nav está construido con flexbox y proporciona una base sólida para contruir todo tipo de componentes de navegación. Incluye algunas modificaciones de estilo (para trabajar con listas), algo de relleno de enlaces para áreas de impacto más grandes y estilo básico desabilidato.

Conclusión: este mejor y perfecto para usarlo como aside (barra de navegación interna)
